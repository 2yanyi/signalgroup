# Ding Chat
Linux GNOME 钉钉 客户端

<br>

## 快速开始
1. [下载 DingChat](https://github.com/yan-xz/dingchat/releases/download/dingchat/dingchat-linux-x64.tar.xz)
2. 解压 dingchat-linux-x64.tar.xz
3. 进入 dingchat-linux-x64/dist
4. 执行
```
sudo sh install
```
